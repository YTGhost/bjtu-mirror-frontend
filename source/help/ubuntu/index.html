Ubuntu 的软件源配置文件是 /etc/apt/sources.list。将系统自带的该文件做个备份，将该文件替换为下面内容，即可使用软件源镜像。

    {% for item in items %}
  1. {{ item.tag }} 
    # 默认注释了源码镜像以提高 apt update 速度，如有需要可自行取消注释
deb https://mirror.bjtu.edu.cn/ubuntu/ {{ item.name }} main restricted universe multiverse
# deb-src https://mirror.bjtu.edu.cn/ubuntu/ {{ item.name }} main restricted universe multiverse
deb https://mirror.bjtu.edu.cn/ubuntu/ {{ item.name }}-updates main restricted universe multiverse
# deb-src https://mirror.bjtu.edu.cn/ubuntu/ {{ item.name }}-updates main restricted universe multiverse
deb https://mirror.bjtu.edu.cn/ubuntu/ {{ item.name }}-backports main restricted universe multiverse
# deb-src https://mirror.bjtu.edu.cn/ubuntu/ {{ item.name }}-backports main restricted universe multiverse
deb https://mirror.bjtu.edu.cn/ubuntu/ {{ item.name }}-security main restricted universe multiverse
# deb-src https://mirror.bjtu.edu.cn/ubuntu/ {{ item.name }}-security main restricted universe multiverse

# 预发布软件源，不建议启用
# deb https://mirror.bjtu.edu.cn/ubuntu/ {{ item.name }}-proposed main restricted universe multiverse
# deb-src https://mirror.bjtu.edu.cn/ubuntu/ {{ item.name }}-proposed main restricted universe multiverse
    2. {% endfor %}
